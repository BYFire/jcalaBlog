####分支:
1. master主分支： 基于docker方式部署，可以动态更换头像，可以上传图片，用nginx做反向代理服务器。
2. simple分支：基于jar包形式部署，不可以动态更改头像，不可以上传图片，不用niginx做反向代理服务器。
